 # Parsedantic

Initial project skeleton for Parsedantic, a Pydantic-native wrapper around the parsy parser
combinator library.

This repository currently contains only the Step 1 setup:

- Project metadata and dependencies in `pyproject.toml`
- Minimal package layout under `src/parsedantic/`
- Basic pytest configuration and a smoke test

Subsequent steps will add the actual parsing functionality, models, and examples.
